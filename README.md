---

# Project Vision

This project explores whether quantum-mechanical and electrochemical properties of molecular systems can be analyzed computationally in order to better understand structural differences between healthy and cancerous cells.

The long-term goal is the conceptual investigation of selective membrane destabilization mechanisms based on charge distribution, molecular symmetry, and quantum-electronic behavior.

This repository documents:
- simulations
- theoretical models
- quantum computing experiments
- reproducible computational workflows
